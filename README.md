# NovaFlow – Responsive Landing Page

A modern and responsive landing page developed for **NovaFlow**, a fictional technology company. The project demonstrates responsive web design, clean UI implementation, and frontend development using HTML, CSS, and JavaScript.
Live link - https://novaflow-app.netlify.app/

## Overview

The website features a professional corporate layout with a focus on usability, accessibility, and responsive design. It is optimized to provide a consistent user experience across desktop, tablet, and mobile devices.

## Features

* Responsive layout for all screen sizes
* Modern and clean user interface
* Sticky navigation bar
* Smooth scrolling and interactive elements
* Well-structured and maintainable code
* Cross-browser compatible design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/novaflow-landing-page.git
```

Navigate to the project directory:

```bash
cd novaflow-landing-page
```

Open `index.html` in your preferred web browser.

## Purpose

This project was developed to strengthen frontend development skills by building a professional, responsive landing page using core web technologies without relying on external frameworks.
